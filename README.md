# Estudos para CPA 2026
O objetivo deste repositório é resumir os principais temas que devem ser cobrados na nova CPA.

## Sobre a prova
É um exame de 50 questões de múltipla escolha a serem resolvidas em no máximo 2h30. É necessário 70% de acerto, ou seja, 35 questões.
